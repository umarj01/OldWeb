Nmap Guide for Beginners

🔍 What is Nmap?

Nmap (Network Mapper) is an open-source tool used for network discovery and security auditing. It’s widely used by cybersecurity professionals and ethical hackers to scan hosts and discover services, ports, operating systems, and vulnerabilities.

🔧 How to Install Nmap

📥 Linux (Debian/Ubuntu)

sudo apt update
sudo apt install nmap

📥 Kali Linux

Already pre-installed.

📥 Windows

Download from the official website: https://nmap.org

📥 macOS (with Homebrew)

brew install nmap

🧪 What are Scan Types?

  • SYN Scan (-sS) – Stealthy and fast
  • TCP Connect Scan (-sT) – Uses full TCP handshake
  • UDP Scan (-sU) – Finds open UDP ports
  • Ping Scan (-sn) – Checks if a host is up
  • Aggressive Scan (-A) – Full system discovery
  • Version Detection (-sV) – Detects service versions
  • OS Detection (-O) – Identifies OS type

🤝 What is TCP Handshake?

The TCP Handshake is a three-step process to establish a connection:

  1. SYN – Client requests connection
  2. SYN-ACK – Server acknowledges and responds
  3. ACK – Client confirms, connection is made

💻 Basic Commands of Nmap

nmap <target>               # Simple scan
nmap -sS <target>           # SYN scan
nmap -sV <target>           # Detect service versions
nmap -O <target>            # OS detection
nmap -A <target>            # Aggressive scan
nmap -p 80 <target>         # Scan specific port
nmap 192.168.1.1/24         # Scan entire subnet
